혜택3.
사용후기 이벤트 : 스타벅스 아메리카노 Tall Size 기프티콘 2매 증정 안내

  • [사용 후기 이벤트 : 응모 방법]
    • - Step 1 행사 기간 내 Galaxy Buds2 구입하고 사용 인증샷 찍기
    • - Step 2 갤캠스 [제품 구매후기 게시판]에 사용 후기 올리기
    • - Step 3 이벤트 안내 게시글 내 [Galaxy Buds2 사용 후기 이벤트 응모하기] 클릭 후, 신청 정보 작성
    • - Step 4 개별 문자 발송을 통한 당첨자 발표
  • 지급기준 : 구매 제품 1개당 1개의 리뷰 작성 가능, 다수 제품 구매 시 구매 수량 만큼의 후기 작성이 가능하며 기프티콘은 중복 증정됩니다.
    • Ex 1) 버즈2 3대 구매 + 사용 후기 3건 작성 -> 사용 후기 3건 모두 인정 *기프티콘 2매 = 기프티콘 총 6장
    • Ex 2) 버즈2 3대 구매 + 사용 후기 1건 작성 -> 사용 후기 1건 인정 *기프티콘 2매 = 기프티콘 총 2장
    • Ex 3) 버즈2 3대 구매 + 사용 후기 4건 작성 -> 사용 후기 3건만 인정 *기프티콘 2매 = 기프티콘 총 6장
  • 사용 후기 이벤트는 사전 판매 응모자 포함 선착순 9,000대에 한해 증정됩니다.
